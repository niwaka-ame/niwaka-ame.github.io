Getting Raiden 2 working on Retropie

I used to be a big fan of Raiden and was able to get Raiden II working on my Retropie during the pandemic. Today, after several some weeks of hard work, I feel like playing this classic again, but I forgot which ROM and emulator I used before.

Tonight after some trial and error I find this ROM (Raiden II, US, Set 1) and it runs on the lr-fbneo emulator on Retropie. The only thing I need to do is to copy this ROM to this directory 

~/Retropie/roms/fba

and the game should work out of the box!
