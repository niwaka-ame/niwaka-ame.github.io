Welcome to my personal blog.

I have a PhD in quantitative biology at the University of Edinburgh and am most interested in microbiology and computational biology.

I use GNU Emacs daily and this blog was edited and built with Orgmode in Emacs. In this blog, I mostly write about how I use my computer, and sometimes happy moments in my life. You’ll see all my Posts and Softwares in this page.

This is a simple, static blog which doesn’t support comments, but please feel free to contact me at: yhuo at tuta dot io.

Softwares

  • nunchaku: (Python) Optimally partitioning data into piece-wise linear segments with Bayesian statistics. Available on PyPI. See also our preprint on BioRxiv.
  • sgd-lookup.el: (Emacs Lisp) An Emacs API wrapper for Saccharomyces Genome Database (SGD).
  • emacs-gene-tools: (Emacs Lisp) An interactive UI to query gene info in a given organism in Emacs (for now works on budding yeast only).
  • metaplate: (Python) Automatically parse the metadata of many plate reader experiments and fuzzy-search for the desired datasets.
