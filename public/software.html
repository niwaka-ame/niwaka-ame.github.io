Software

  • nunchaku: (Python) Optimally partitioning data into piece-wise linear segments with Bayesian statistics. Available on PyPI. See also our preprint on BioRxiv.
  • sgd-lookup.el: (Emacs Lisp) An Emacs API wrapper for Saccharomyces Genome Database (SGD).
  • emacs-gene-tools: (Emacs Lisp) An interactive UI to query gene info in a given organism in Emacs (for now works on budding yeast only).
  • metaplate: (Python) Automatically parse the metadata of many plate reader experiments and fuzzy-search for the desired datasets.
